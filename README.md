## üèó Setup
Follow the below steps to setup this repo locally.

1. Fork and clone the repo.
2. Install dependencies.
```
    npm i
```
3. Create a .env file and paste the below variables with their values.
```
AUTH_SECRET=
MONGODB_URI=
GEMINI_API_KEY=
```
4. Run the server
```
    npm run dev
```
